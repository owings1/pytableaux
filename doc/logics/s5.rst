    Universal Access
    ----------------

    S5 includes both the *reflexivity* restriction of :ref:`T <T>`, and the
    *transitivity* restriction of :ref:`S4 <S4>`:

    .. include:: include/t/m.reflexivity.rst

    .. include:: include/s4/m.transitivity.rst

    S5 adds an additional *symmetric* restriction on the access relation for models:

    .. cssclass:: definiendum smallcaps

    Symmetry

    .. cssclass:: definiens

    In every model, for each world :m:`w` and each world :m:`w'`, if :m:`<w,w'>`
    is in the access relation, then so is :m:`<w',w>`.

    This is witnessed in the tableaux system through the :ref:`Symmetric Rule
    <symmetric-rule>`.
