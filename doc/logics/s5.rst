.. _S5:

**********************************
L{S5} - S5 Normal Modal Logic
**********************************

L{S5} is an extension of {@K} with a reflexive, symmetric, and transitive
access relation. The access relation provides `Universal Access`, meaning
all worlds are visible to all worlds.
