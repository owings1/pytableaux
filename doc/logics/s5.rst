    Logical Consequence
    ===================

    **Logical Consequence** is defined just as in :ref:`K` :

    - *C* is a **Logical Consequence** of *A* iff all models where the value of *A* is **T**
      at *w0* are models where *C* also has the value **T** at *w0*.
